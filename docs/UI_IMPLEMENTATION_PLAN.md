# 印刷报价系统 - 前端UI复刻实施计划

基于 yinshuabaojia.com 的视觉风格和布局结构参考，用原创资源实现（不复制品牌 logo、专有图片、文案）

## 一、整体布局架构

### 1. 顶部导航栏
- **位置**: 固定顶部
- **内容结构**:
  - 左侧: 系统标题 "报价软件功能演示报价系统"（改为我们自己的品牌名）
  - 右侧: 用户信息区
    - 用户ID显示（如 "1708888"）
    - "修改密码" 链接
    - "安全退出" 链接
- **颜色方案**: 深色背景（观察为深蓝/深灰色系）

### 2. 主导航菜单
- **位置**: 顶部导航栏下方
- **菜单项**: 首页 | 报价 | 彩盒 | 管理 | 参数 | 通知
- **样式**: 横向排列，鼠标悬停高亮
- **布局**: 左对齐，等间距

### 3. 面包屑导航
- **位置**: 主导航下方
- **格式**: 首页 > 专版联单报价
- **样式**: 轻量级，灰色文字，箭头分隔

### 4. 主内容区布局
- **结构**: 左右分栏
  - **左侧**: 表单输入区（约占 40-45%）
  - **右侧**: 报价结果展示区（约占 55-60%）

## 二、无碳联单表单区（左侧）

### 表单容器样式
- **背景**: 白色卡片
- **边框**: 1px 实线，浅灰色
- **圆角**: 4-6px
- **内边距**: 20px
- **阴影**: 轻微阴影（box-shadow: 0 2px 8px rgba(0,0,0,0.1)）

### 表单字段列表

#### 1. 成品尺寸
- **标签**: "成品尺寸" (左对齐，加粗)
- **控件**: 
  - 下拉选择框（宽度约 200px）
  - 选项: 72开(88×128) | 64开(105×145)A6 | 48开(210×95) | 32开(210×140)A5 | 16开(210×285)A4 | 8开(420×285)A3 | 正度系列
  - 右侧勾选框: "自定义尺寸"

#### 2. 订单数量
- **标签**: "订单数量"
- **控件**: 数字输入框 + 单位文字 "本"
- **默认值**: 100

#### 3. 联数
- **标签**: "联数"
- **控件**:
  - 下拉选择框（二联 | 三联 | 四联 | 五联 | 六联）
  - 右侧勾选框: "显示每联详情"

#### 4. 每本页数
- **标签**: "每本页数"
- **控件**: 数字输入框 + 动态计算显示 "页( XX份)"
- **计算逻辑**: 份数 = 页数 / 联数

#### 5. 印刷颜色
- **标签**: "印刷颜色"
- **控件**: 
  - 第一行: "正面:" 下拉框（空白 | 单黑 | 双色 | 四色(彩色) | 1-4专色 | 彩色+专色组合）
  - 第二行: 勾选框 "背面:" + 复制正面选项的下拉框
- **附加选项**（第二行，多个勾选框）:
  - 正面大实地 | 背面大实地 | 正面高品质 | 背面高品质

#### 6. 纸张克重
- **标签**: "纸张克重"
- **控件**: 下拉选择框（50克 | 80克 | 108克）

#### 7. 后道工序
- **标签**: "后道工序"
- **控件**: 多行勾选框布局（2列网格）
  - 第1行: 我要设计 | 装订（二级勾选: 胶左/胶头）| 印张交货
  - 第2行: 加卡纸 | 打号码
  - 第3行: 压痕压点线 | 加封面
  - 第4行: 印封面 | 换边联字
  - 第5行: 打包
  - 最后一行: "其他后工:无"（文本显示）

### 底部操作区

#### 客户信息输入
- **客户**: 文本输入框（带提示 "请选择或输入客户"）
- **产品名**: 文本输入框（带提示 "请输入产品名"）

#### 操作按钮组
- **多数量**: 文本链接
- **勾选框**: "自填利润率"
- **主按钮**: "自助报价"（蓝色背景，白色文字，较大尺寸）
- **状态提示**: "计算完成！"（动态显示）
- **次按钮**: "ERP下单"（灰色背景）

## 六、交互细节

### 1. 表单交互
- **输入框聚焦**: 蓝色边框高亮（2px solid #3498db）
- **下拉框**: 点击展开动画（200ms ease-out）
- **勾选框**: 勾选时显示蓝色对勾动画
- **禁用状态**: 灰色背景 + 不可点击光标

### 2. 按钮交互
- **悬停**: 背景颜色加深 10%，过渡时间 200ms
- **点击**: 轻微缩小效果（scale: 0.98）
- **加载中**: 显示旋转加载图标 + 按钮文字变为"计算中..."

### 3. 表格交互
- **行悬停**: 整行背景变为 #f5f7fa
- **可编辑单元格**: 双击进入编辑状态
- **数字格式化**: 自动添加千分位逗号

### 4. 实时计算
- **触发时机**: 任何表单字段变化后 500ms（防抖）
- **加载状态**: 右侧结果区显示骨架屏
- **错误提示**: 表单上方显示红色提示条
